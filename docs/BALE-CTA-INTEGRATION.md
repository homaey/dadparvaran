# اتصال CTA به صفحات سایت

کامپوننت آماده:

```tsx
import { BaleConsultationCTA } from "@/components/consultation/BaleConsultationCTA";

<BaleConsultationCTA locale="fa" />
```

برای نسخه انگلیسی:

```tsx
<BaleConsultationCTA locale="en" />
```

پس از ادغام، ایجنت باید CTA را با رعایت طراحی موجود در این نقاط قرار دهد:

- Hero صفحه اصلی، به‌عنوان CTA اصلی یا ثانویه طبق تصمیم محصول
- انتهای صفحات خدمت
- انتهای مقاله‌ها
- صفحه تماس
- پروفایل وکیل
- دکمه شناور موبایل، بدون پوشاندن عناصر مهم

CTA باید به `NEXT_PUBLIC_BALE_BOT_URL` هدایت شود. در معماری فعلی لینک گفت‌وگوی بازو توصیه شده تا کاربر ابتدا بازو را Start کند و سپس Mini App را از دکمه داخل بازو باز کند؛ این کار امکان ارسال پیام واگذاری به کاربر را پایدارتر می‌کند.
