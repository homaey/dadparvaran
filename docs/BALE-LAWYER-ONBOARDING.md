# فعال‌سازی حساب بله وکیل

1. مدیر از API یا UI مدیریت، برای TeamMember یک کد فعال‌سازی می‌سازد:
   `POST /api/admin/lawyers/{id}/bale-activation`
2. وکیل بازو را Start می‌کند.
3. وکیل در چت خصوصی بازو می‌فرستد: `/link CODE`
4. بازو `from.id` را به پروفایل وکیل متصل می‌کند.
5. مدیر لینک عمومی حساب کاری وکیل را بررسی و `isVerified` را فعال می‌کند.
6. وکیل باید عضو گروه خصوصی باشد و ظرفیت/دسترس‌پذیری فعال داشته باشد.

## توصیه عملی

برای هر وکیل یک حساب کاری بله با نام و تصویر حرفه‌ای ایجاد شود. لینک عمومی حساب باید دستی آزمایش شود. از حساب خانوادگی یا حساب فاقد Username عمومی استفاده نشود.
