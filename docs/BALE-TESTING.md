# برنامه تست

## تست خودکار

```bash
npm ci
npx prisma generate
npm run typecheck
npm test
npm run build
```

تست‌های همراه بسته:

- Parser دکمه‌های Callback
- اعتبارسنجی امضای Mini App و انقضای auth_date
- State transition policy
- موفقیت/خطای Bale API Client با Fetch Mock

## تست دستی Staging

1. فرم را از داخل بله باز کنید؛ بازشدن مستقیم مرورگر باید رد شود.
2. درخواست ثبت و کد عمومی نمایش داده شود.
3. پیام گروه فاقد نام، تلفن و شرح کامل باشد.
4. دو وکیل تقریباً هم‌زمان دکمه پذیرش را بزنند؛ فقط یک نفر موفق شود.
5. پیام گروه ویرایش و دکمه حذف شود.
6. وکیل منتخب کارت خصوصی دریافت کند.
7. کاربر لینک دقیق حساب همان وکیل را دریافت کند.
8. دکمه «ارتباط برقرار شد» وضعیت را در داشبورد تغییر دهد.
9. یک Webhook تکراری دوباره اثر نگذارد.
10. قطع موقت API بله باعث حذف رکورد درخواست نشود.
